Mixstream Pro Shift Key Quick reference for live DJ use

Numark Mixstream Pro Shift Shortcuts

A mobile-friendly cheat sheet combining the original Shift list plus Shift + Auto Loop and Shift + Roll.

Live DJ tip: use this page before the gig and during setup. During a live mix, the most useful shortcuts are usually Shift + Sync, Shift + Cue, Shift + Roll, and Shift + pads for deleting cues/loops or stopping samples.
Shift + SyncTurn Sync off
Shift + CueJump to track start
Shift + RollSampler mode
Shift + PadClear cue / loop / sample
Screen / Browse
Shift+Browse Knob Press

Add selected track to the Prepare list.

Use this when you find a possible next song and want to keep it ready without loading it to a deck.

Screen / Browse
Shift+View

Change the waveform / screen view.

Useful when you want a different deck display or waveform layout while mixing.

Screen / Browse
Shift+Source / Lighting

Open lighting controls on supported firmware/layouts.

Use this for Engine Lighting / SoundSwitch lighting control when your lighting setup is configured.

Must-know
Shift+Smart Scratch

Switch between Smart Scratch and Jog mode behaviour.

Helpful when you want the jog wheel to behave more like pitch bend / nudge instead of scratch control.

Must-know
Shift+Sync

Turn Sync off.

Very useful when you want to stop BPM sync and take manual tempo control again.

Must-know
Shift+Cue

Return to the beginning of the track.

Good for quickly restarting from the start point while preparing or practising.

Must-know
Shift+Play / Pause

Stutter-play from the initial cue point.

Use carefully as a performance restart/stutter effect.

Must-know
Shift+Pitch Bend − / +

Change the pitch fader range.

Useful when you need smaller or larger tempo control, such as ±4%, ±8%, ±10%, ±20%, ±50% or ±100%.

Pads
Shift+Hot Cue Pad

Clear / delete that hot cue.

Use this to remove an unwanted cue point from the current track.

Pads
Shift+Saved Loop Pad

Delete that saved loop.

Use this when a saved loop is wrong or no longer needed.

Careful
Shift+Auto Loop

Usually no separate Shift shortcut is documented for Auto Loop.

For Auto Loop, press Auto Loop to enter the mode, then press Auto Loop again for the second bank of loop lengths. Press a pad to activate/deactivate the loop.

Keep this row here as a reminder: if you are trying to access more Auto Loop lengths, press Auto Loop again rather than relying on Shift.
Pads
Shift+Roll

Enter Sampler Mode on Engine DJ 3.0+.

Use this to trigger drops, drums, risers, FX or other one-shot samples from the performance pads.

Sampler tip: in Sampler Mode, holding Shift + a playing sample pad stops that sample. In library view, Shift + sample pad can unload/eject a sample slot.

Note: Exact behaviour can vary slightly by firmware version and controller model variant. Test these shortcuts before using them live.
